Why having fun when machines can do it for us ?

Thired of having fun solving sudoku grids ? Here is a sudoku resolver with 3 options :

  • JS : The algorithm will be wirtten in JavaScript, by far the slowest option. Your machine will do the calculations to solve the grid.
  • WASM : The algorithm will be written in Rust and compiled in WASM. About has fast as the API. Your machine will do the calculations to solve the grid.
  • API : The algorith will be written in Rust and will get and return the grid via HTTP. About has fast as WASM if you have a good internet connexion, else, it will be slower. My server will do the calculations, please do not abuse it or I will have to stop it.
